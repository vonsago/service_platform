# service_platform

docker dashboard 

### 环境准备
    需要有docker环境支持
    下载项目中的docker-compose.yml文件

### 并在yml文件所在目录运行命令
    docker-compose up -d

### 本地运行
`python3 app/app_runner.py`

### 运行前端
同时本地运行[前端UI](https://github.com/vonsago/DDP_UI)
